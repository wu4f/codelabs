Regular expressions allow one to perform advanced string matching operations. The language itself can be difficult to learn. LLMs have the potential for obviating the need for learning the syntax of a regular expression.

Query your LLMs to produce commands using Linux's grep command (global regular expression print) that searches words in a file called rockyou.txt that:

  • Start with abcde followed by an optional f at the end.
  • Start with abcde followed by one or more f characters at the end
  • Start with abcde followed by zero or more f characters at the end
  • Start with 19 instances of the letter q
  • Matches all 10 character words that begin with 3 letters of a and end with 3 letters of a
  • Matches words that consist of only alphabetic characters
  • Matches words that are made up of exactly 6 digits
  • Matches words that consist of all non-alphanumeric characters
  • Matches words that begin with o, ends with 4 numbers, and has a > character in it
